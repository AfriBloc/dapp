## Pull Request: done working on afribloc project

This pull request includes the latest work on the afribloc project, focusing on user profile and wallet functionalities, along with several UI components and utility hooks.

**Summary of Changes:**

*   **User Profile Enhancements:**
    *   Added functionality for changing user passwords.
    *   Introduced a personal information display box.
*   **User Wallet Management:**
    *   Implemented features for funding user wallets.
    *   Added functionality for withdrawing funds.
    *   Included a transaction history view.
    *   Developed a wallet balance display card.
*   **New UI Components:**
    *   Updated `dialog.tsx`, `radio-group.tsx`, and `select.tsx` components.
    *   Created a new `select-field.tsx` for forms.
*   **New Utility Hooks:**
    *   Added `use-change-password.ts` for password management.
    *   Added `use-fund-wallet.ts` and `use-withdraw.ts` for wallet operations.
    *   Introduced `use-clipboard.ts` for clipboard interactions.
*   **New SVG Assets:**
    *   Added `bank.svg`, `card.svg`, `check.svg`, `hashpack.svg`, `stripe.svg`.
